handle_realtime_connection

Function to connect the client to socket server.

once any user created a new event the ping_event_added socket will be triggered to all the connected clients. All client will listen to the ping_event_added and will ping back to the sync_event_added socket server with the client token and server will call back ping_event_recived to the connected client.
